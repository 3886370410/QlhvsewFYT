# Python航空数据分析系统

## 核心功能
- 登录注册
- 航班数据管理
- 分页查询
- 个人信息修改
- 用户管理
- 不同航空公司航班数据分析
- 不同出发城市航班数据分析
- 不同航班机型数据分析
- 平均准点率分析
- 平均误点率分析
- 不同出发省份航班分析
- 到达多个城市分析

## 适用场景
- 航空公司运营管理
- 航空行业数据分析研究
- 航班性能评估

## 技术特点
- 使用 Flask 构建 Web 应用
- 基于 MySQL 数据库进行数据存储
- 前端界面采用 HTML、CSS、JS 和 layui
- 系统可运行，无需额外安装依赖
